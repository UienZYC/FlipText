# FlipText
Flip your text, flip the language. An in-place translator for any input field.

场景：输入框内打字后，F1将中文翻译成英语，Tab替换或插入英语

翻译功能对于能按下Ctrl+C复制的大多数场景生效

# 功能
双击exe文件运行后，可在任务栏查看到小图标

按F1优先对鼠标选中文本进行中英互译，若无选中，则自动选取光标所在行，并进行翻译

译文弹窗显示在鼠标所在位置

查看翻译后，可按Tab，自动将选区替换为译文，若无选中，则将光标所在当前行替换为译文。

查看翻译后，可移动光标至其他任意可输入文本区域，按下Tab，将译文黏贴到光标处。

Esc关闭弹窗

任务栏右键Exit退出

# 注意
本软件由 AutoHotkey 编写，可能会被杀毒软件误报

# 注意
不应在重要文件中未经测试地使用此脚本

不应在未经测试的情况下，在表格文件中使用此脚本
